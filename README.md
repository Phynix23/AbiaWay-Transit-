# AbiaWay Transit System

Smart transit system for Abia State with real-time tracking, digital payments, and route planning.

## Features
- 🗺️ Live bus tracking on interactive map
- 💳 Digital wallet with NFC payment simulation
- 🚌 Route planning and booking
- 👨‍✈️ Driver dashboard with GPS simulation
- 🤖 AI travel assistant
- 📱 PWA ready

## Tech Stack
- React 19
- TailwindCSS
- Leaflet Maps
- Vite
